# 皮皮虾医生 用户手册

## 它能做什么

皮皮虾医生可以帮你：

- 看你的 OpenClaw 系统是否健康
- 看最近日志有没有明显错误
- 把报错翻译成小白能懂的解释
- 给出安全修复计划
- 查询上次类似问题怎么处理
- 把处理结果写成病历，方便复用

## 常用说法

```text
皮皮虾医生 体检
皮皮虾医生 报错了：fetch failed timeout
皮皮虾医生 帮我修一下：unknown tool
皮皮虾医生 上次这个问题怎么处理
皮皮虾医生 带我上手
```

## 重要提醒

皮皮虾医生不会直接乱修。只读检查可以直接跑；涉及写文件、安装依赖、授权、重启或改配置时，会先给你计划并等你确认。
